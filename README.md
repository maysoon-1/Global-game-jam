# Global-game-jam 🎮🪄
### Global Game Jam is an annual event where participants worldwide create  games in a short time frame, usually over a weekend. Our game was designed under the theme of the hackathon for the year  2023, which is the (ROOT) And left the freedom to express the “ROOT” to the contestants, And we goes With a lonely burned root he was responsible for the reconstruction of the forest.By overcoming obstacles up to the elixir achieved for that

## game building 🕹️
We choose Unity as it is a great tool for prototyping everything from 
games, to interactive visualisationsp .Also is highly flexible platform 
well-documented, and highly extensible to build pretty much any genre 
of game you can think of.
